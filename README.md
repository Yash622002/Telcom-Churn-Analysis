Telco Customer Churn Analysis
Overview

This project analyzes customer churn using the Telco dataset. It includes data cleaning, exploratory data analysis (EDA), and visualizations to identify key churn patterns.

Key Insights

Around 26.5% of customers churned.

Month-to-month contracts and electronic check payments had the highest churn.

Senior citizens and customers without add-on services were more likely to leave.

Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn
